Some of My Work

Pearish (In Development)

Led & managed an open source project to create a digital pantry app. Using image recognition and language processing, the app scans receipt images to detect food keywords. The results are compared to a database to match keywords and stored in a table for each user.

Developed using Python, React, React-Table, NGINX, Spacy NLP, OCR.space and MongoDB. Containerized with Docker

DotaBot

Created a bot for Discord, a messaging and voice application program. Used by 200+ users, the bot has custom commands to generate character guide links and create a profile associating Discord with Steam video game platform.

Developed using Python, Discord.py, MongoDB, containerized with Docker and hosted on Heroku.

Movie Reminder (Freelance Job)

Made an automated notification script for movie recording, saving 30 minutes daily.

A custom database was made, compared to today's movies scraped from a website and sent automatic emails if a movie required recording.

Developed using Python, Python-Docx, EZGmail and hosted on Amazon EC2 Linux Instance
